Criando um projeto novo

Dentro do diretorio do projeto de seguinte comando:
npm init -y

depois o comando - configura o projeto
npm install express

depois cria o arquivo index.js 
feito isso o projeto já está criado!

*****************************************************************************

OBS:O npx executa um pacote sem baixar!
Para criar um app usando react, o comando é:
    npx create-react-app nome_do_projeto

comando para não ter que ficar recarregando a aplicação.
npm install nodemon -D 
-D significa que o nodemon será instalado e apenas para
desenvolvimento da aplicação. 

Apos executar esse comando, é necessario ir até o aquivo 
packege.json e alterar o script "test" por "start" e colocar
no seu argumento "nodemon index.js". Depois é só dar o comando:

npm start

fazendo isso o nodemon ira
executar o index.js sem a necessidade de reiniciar o servidor
toda vez que precisar modificar alguma coisa na aplicação!

Para trabalhar banco de dados utilizando o javaScript é necessario
instalar uma ferramenta chamada knex, para isso execute o comando

npm install knex

nesse projeto foi usado o banco de dados sqlite, para isso é necessario
o seguinte comando:

npm install sqlite3

agora para a conexão do banco de dados com a aplicação o comando é o
seguinte:

npx knex init

esse comando vai criar o arquivo knexfile.js que ira ter as configurações
de acesso ao banco de dados!

criando migrate para banco de dados, utilizando o comando:
npx knex migrate:make nome_da_migrate
quando é criada, aparece um warning, basta copira useNullAsDefault
e colar no arquivo knexfile.js em development, dessa froma: 
useNullAsDefault: true,

dentro da migrate, criamos a tabela dentro de: exports.up = function(knex), 
colocamos o seguinte codigo:

knex.schema.createTable('ongs', function(table){
});

com isso criamos a tabela ongs! Agora vamos criar os campos da tabela:

exports.up = function(knex) {
 return knex.schema.createTable('ongs', function(table) {
    table.string('id').primary();
    table.string('nome').notNullable();
    table.string('email').notNullable();
    table.string('whatsapp').notNullable();
    table.string('city').notNullable();
    table.string('uf', 2).notNullable();
  });
};

por ultimo devemos implementar o medoto down:
exports.down = function(knex) {
  return knex.schema.dropTable('ongs');
};

para o arquivo de banco de dados da tabela devemos rodar o seguinte comando:
npx knex migrate:latest
